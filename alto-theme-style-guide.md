# Alto 主题样式详解

> 基于仓库默认配置（`alto.css` + `alto/alto.css`，母主题默认引用 Lapis；可切换为 Spring 或暗色 `alto-dark.css`）。这里按 Typora 中常见文本类型说明实际呈现效果，并给出示例。

## 主题结构与配色
- **界面加壳**：`alto.css` 设置主色（`--alto-accent-color` 默认 #487dac）、侧边/背景（`--alto-main-color` 默认淡蓝）与内容底色（`--alto-content-color` 白色），并在 `content` 区域提供圆角与悬浮阴影。
- **母主题**：默认 `alto/lapis/lapis.css`（蓝色、轻量衬线风格）；可改为 `alto/spring/spring.css`（绿色、动效丰富）；暗色用 `alto-dark.css`（导入 `lapis-dark.css`，深灰背景、浅色文字）。
- **字体**：正文/标题使用 Cantarell + 思源宋体，等宽元素用 JetBrains Mono。

## 正文与标题
- **排版**：正文宽度约 950px，行高 1.6，两端对齐。正文颜色 #40464f（暗色模式下 #e4e4e4）。
- **标题层级**：  
  - `h1` 居中、2rem，底部留白较大。  
  - `h2` 带胶囊色块背景（使用主色反白显示），行内展示；`h3-h6` 常规行内，无背景。  
  - 标题全部用宋体，颜色取主色，`strong` 亦继承主色。

## 链接、强调与高亮
- 链接为主色带下划线，`strong` 着主色，`em` 仅略微加内边距。  
- `mark` 高亮色为淡黄（暗色下文字反色显示），下划线偏移以保持视觉整洁。

## 列表与任务
- 列表标记加粗、颜色为浅蓝系 `--marker-color`。`li` 行距 1.7rem，层级缩进 20px。  
- 任务列表复选框用圆角描边，选中后显示主色对勾。

## 引用与提示块
- 引用块/提示块左侧 3px 主色竖线，内边距 15-30px，浅灰背景（暗色为深灰）。  
- Typora 内置 `[!NOTE]/[!TIP]/[!WARNING]/[!CAUTION]/[!IMPORTANT]` 等 alert：左线和文字按语义配色（蓝/绿/黄/红/紫），背景对应淡色填充。

## 代码
- **行内代码**：圆角 3px、浅灰底，文字为主色，等宽字体。  
- **代码块**：圆角 10px，浅灰底（暗色深色），行号分隔线淡蓝；CodeMirror 语法高亮：关键字青蓝、字符串紫、数字深蓝，选中为半透明灰。暗色方案替换为霓虹蓝/绿配色。

## 表格
- 100% 宽度，边框 1px 浅灰，单元格 5px×10px 内边距；表头居中、加粗宋体、主色文字。悬停行/列使用块背景高亮。

## 图片、脚注、分隔线
- 图片居中、自适应宽度，唯一图片时带投影。  
- 脚注编号浅灰底、主色文字；数学行内公式同样用主色。  
- 分隔线为 2px 圆角浅灰。  
- 复选框、脚注、数学、表格在暗色主题下同步反色处理。

## 侧边栏与界面元素
- 侧边栏透明叠在主背景上，悬停/选中文件卡片缩放并加浅色描边，文件夹图标主色。  
- 大纲：`h1` 加粗主色；`h2` 圆点 + 竖线指引；当前条目右侧显示主色箭头。  
- 滚动条窄 0.5rem，半透明；菜单/弹窗/搜索面板均用圆角 + 浅色卡片，悬浮时主色描边。  
- 暗色下内容卡片与菜单底色改为深灰，描边与 hover 色同步降低亮度。

## 快速示例（直接粘贴到 Typora）
```markdown
# 一级标题（居中）
## 二级标题（色块反白）
### 三级标题

普通段落，两端对齐，含 **粗体**、*斜体*、==高亮==、以及 [链接示例](https://example.com)。

- 无序列表一级
  - 二级项（标记加粗上色）
1. 有序列表
2. 支持任务列表：
   - [ ] 未完成
   - [x] 已完成

> [!TIP]
> 引用/提示块左侧主色竖线，浅灰底。

内联代码 `console.log('hello')` 与下方代码块：

```js
function greet(name) {
  console.log(`Hello, ${name}!`);
}
greet('Alto');
```

| 表头A | 表头B |
| --- | --- |
| 单元格1 | 单元格2 |
```

---
如需切换母主题，将 `alto.css` 中的 `@import "alto/lapis/lapis.css";` 改为 Spring 或其他主题，并调整 `:root` 颜色变量；暗色使用 `alto-dark.css`。
